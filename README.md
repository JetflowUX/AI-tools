# 🤖 AI Tools Directory

A Vue 3 application that lists AI tools organised by category. Features a searchable, filterable directory with 30+ tools across 6 categories.

## Features

- **6 categories**: Text & Writing, Image Generation, Code & Development, Audio & Video, Productivity, Research & Data
- **30 curated AI tools** with descriptions, links, and tags
- **Live search** — filter tools instantly by name, description, or tag
- **Category navigation** — jump to any category with sticky header links
- **Dark mode** — automatically follows system preference
- **Responsive layout** — works on all screen sizes

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- [Vue 3](https://vuejs.org/) (Composition API + `<script setup>`)
- [Vite](https://vite.dev/) — fast dev server and build tool
